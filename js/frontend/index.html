Sintaxis JavaScript

  • Valores
    • Literales
    • Variables
      • Variable no inicializadas tienen el valor 'undefined'
      • Declaraciones multiples separadas por coma, con inicialización de ser requerido
  • Operadores
    • Aritmeticos
      • + - * / % ++ --
      • Concatenación de cadenas de caracteres con + y +=
      • Los numeros al sumarse con cadenas de caracteres son convertidos a cadenas
    • Asignación
      • = += -= *= /= %=
    • Comparación
      • < > <= >= == === != !==
      • El operador == y != verifica solo valor, por lo que '5' == 5 se considera true
      • El operador === y !== verifica valor y tipo, por lo que '5' === 5 se considera false
      • Se puede hacer comparaciones con cadenas de caracteres y las mismas seran comparadas caracter a caracter segun su valor ASCII
    • Logicos
      • && (AND), || (OR), ! (NOT)
    • Ternario
      • (CONDICION) ? (TRUE) : (FALSE)
    • Tipo
      • typeof
      • instanceof
